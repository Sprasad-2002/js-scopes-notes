# JavaScript Scopes 📚

This repository contains simple examples and explanations of different types of scopes in JavaScript.

## 📌 Topics Covered

- Global Scope
- Block Scope
- Function Scope
- Arrow Function Context
- Scope Chain
- this Keyword

## 📁 Files

- GlobalScope.html → Demonstrates global scope
- BlockScope.html → Demonstrates block scope
- BlockScope2.html → Additional block scope examples
- FunctionScope.html → Function scope examples
- ArrowFunctionContext.html → Arrow function behavior
- Scopechain.html → Scope chain concept
- thisKeyword.html → Understanding `this`

## 🎯 Purpose

To understand how variable scope works in JavaScript and how execution context behaves.

## 🚀 How to Run

Just open any HTML file in your browser.
